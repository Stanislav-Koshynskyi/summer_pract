# Rules
### Controls
## Game
WASD - player movement  
LMB - shoot  
E - interact  
SHIFT (hold) - run mode  
C (hold) - sneak mode  
G - drop weapon  
ESC - pause
## Menu
LMB - select  
SCROLL - scroll through the character list
### Victory conditions
## Level 1, entrance
Find a way to get to one of the elevators below  
## Level 2, halls
Defeat everyone who gets in your way  
## Level 3, party
Defeat the party champion in one of the rooms
