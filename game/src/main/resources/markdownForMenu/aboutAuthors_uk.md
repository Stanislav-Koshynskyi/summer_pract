### Різник Нікіта 
## Розробник В (Content + Core systems)
Зони відповідальності:
- Створення систем взаємодії: відкриття дверей, підбір зброї, безшумні вбивства ворогів
- Система слуху ворогів (HearingSystem, SoundEventQueue) з урахуванням перешкод (стін та дверей)
- Система випромінювання кроків гравця (FootstepEmitter) відповідно до режимів руху
- Збереження та завантаження ігрового прогресу у JSON форматі (SaveManager)
- Меню вибору рівнів, статистика проходження та кампанія
